Discussion Post: Review of IT Code of Conduct

Review of IT Code of Conduct

Key Documents

The document "Communication: Data protection rules as a trust-enabler in the EU and beyond – taking stock" (European Commission, 2019) talks about the critical role of data protection in forming trust among individuals, businesses and public authorities. The General Data Protection Regulation (GDPR, 2016) outlines data protection laws for the European Union. It establishes the right of individuals regarding personal data. It should be noted that UK organisations no longer need to comply with these documents but with the UK versions of them.

I reviewed the established Code of Conduct for a Sixth Form College in the UK. There is a core Code of Conduct document that is supported by individual documents covering, such as the Internet Acceptable Use Policy, the Setting Passwords Protocol, and the BYOD to Work Policy.

Best Practices

The Data Protection Policy follows the GDPR(UK) and lays out the responsibilities for protecting personal data. A lot of student and teacher data is held, and this document is clear but comprehensive, including procedures for data access requests and a data retention schedule. The Internet Acceptable Use Policy defines what is reasonable behaviour and what actions could compromise the network. This is an important document as it not only applies to teachers but also to students.

Areas for Improvement

When conducting this research, I found that each of the policies and procedures were b but found it difficult to locate them. They are all stored in one folder with all the other college policy documents. A unified Code of Conduct document could improve comprehension of the subjects and therefore compliance. However, retaining separate documents can allow for more detail to be included for each.

Incident Reflection

Recently the sixth form conducted a simulated phishing attack to see the actions of staff. Many staff opened the email and read it, and many opened the attachment that asked for a username and password. The exercise highlighted the need for ongoing staff education and clear reporting protocols.

Future Implications

The sixth form is currently going through the process of gaining Cyber Essentials Certification (National Cyber Security Centre, N.D.) and are currently in the data collection phase. We could also establish cyber security training for new and current staff to highlight the documents and where to find them, giving key details. There was also no formal incident response plan except to tell IT support.

Job Specification

Computer Games Developer

A games developer needs a strong foundation in programming, creativity, and collaboration. Looking through the job listings on Monster Jobs, I found a lot of skills needed to follow a career as a Games Designer.

Skill Type Essential Skills Desirable Skills
Technical

Programming Skills: Proficiency in C++, C# or Python. Essential for writing game mechanics.

Knowledge of Game Engines: Expertise in Unity or Unreal Engine for designing, developing and deploying games.

Version Control: Git and other version control tools are crucial for managing projects.

Graphic Design Skills: Knowledge of tools like Photoshop or Illustrator to create visual assets.

Multiplayer/Networking Development: Multiplayer games are a focus for many studios, requiring experience in network protocols.

Mobile and Console Platforms: This can be valuable for studios that produce cross-platform games.
Cognitive

Problem-Solving Skills: Ability to troubleshoot technical issues, debug code.

Attention to Detail: Ensuring a polished, bug-free game by focusing on every aspect of the game.

Creative Thinking: Developing unique game ideas, stories, mechanics and gameplay features that captivate players.

Adaptability: The capacity to quickly learn new technologies, programming languages and adapt to evolving industry trends.
Leadership

Teamwork: Collaborating with designers, artists, and other developers.

Project Management: Organising development tasks, managing timelines and resources to deliver projects on schedule.
